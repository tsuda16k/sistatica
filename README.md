統計分析用のさまざまな関数を提供するパッケージです。

## インストール方法

以下の手順でインストールできます。
もしpakパッケージをまだインストールしていない場合ははじめにそれをインストールしてください。

``` r
install.packages("pak")
pak::pak("tsuda16k/sistatica")
```
