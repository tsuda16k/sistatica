<!-- badges: start -->

統計分析用のさまざまな関数を提供するパッケージです。

## インストール

以下の手順でインストールできます。
もしpakパッケージをまだインストールしていない場合ははじめにそれをインストールしてください。

``` r
install.packages("pak")
pak::pak("tsuda16k/sistatica")
```

## 使い方

This is a basic example which shows you how to solve a common problem:

``` r
library(sistatica)
## basic example code
```
