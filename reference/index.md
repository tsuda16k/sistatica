# Package index

## All functions

- [`friedmanOmni()`](friedmanOmni.md) : Omnibus test for the Friedman
  test (one-way repeated-measures, nonparametric)
- [`friedmanPost()`](friedmanPost.md) : Post hoc tests for the Friedman
  test (paired Wilcoxon + Holm p-values)
- [`kruskalOmni()`](kruskalOmni.md) : Omnibus test for the
  Kruskal-Wallis test (one-way, nonparametric)
- [`kruskalPost()`](kruskalPost.md) : Post hoc tests for the
  Kruskal-Wallis test (Dunn p-values + Wilcoxon HL CIs)
- [`rmOmni()`](rmOmni.md) : Omnibus test for repeated-measures ANOVA
  (rm-ANOVA)
- [`rmPost()`](rmPost.md) : Post hoc tests for repeated-measures ANOVA
  (paired t-tests)
- [`welchOmni()`](welchOmni.md) : Omnibus test for Welch one-way ANOVA
- [`welchPost()`](welchPost.md) : Post hoc tests for Welch one-way ANOVA
  (Games-Howell p-values + Welch t-test CIs)
