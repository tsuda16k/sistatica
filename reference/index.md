# Package index

## All functions

- [`friedmanOmni()`](https://tsuda16k.github.io/sistatica/reference/friedmanOmni.md)
  : Omnibus test for the Friedman test (one-way repeated-measures,
  nonparametric)
- [`friedmanPost()`](https://tsuda16k.github.io/sistatica/reference/friedmanPost.md)
  : Post hoc tests for the Friedman test (paired Wilcoxon + Holm
  p-values)
- [`kruskalOmni()`](https://tsuda16k.github.io/sistatica/reference/kruskalOmni.md)
  : Omnibus test for the Kruskal-Wallis test (one-way, nonparametric)
- [`kruskalPost()`](https://tsuda16k.github.io/sistatica/reference/kruskalPost.md)
  : Post hoc tests for the Kruskal-Wallis test (Dunn p-values + Wilcoxon
  HL CIs)
- [`rmOmni()`](https://tsuda16k.github.io/sistatica/reference/rmOmni.md)
  : Omnibus test for repeated-measures ANOVA (rm-ANOVA)
- [`rmPost()`](https://tsuda16k.github.io/sistatica/reference/rmPost.md)
  : Post hoc tests for repeated-measures ANOVA (paired t-tests)
- [`welchOmni()`](https://tsuda16k.github.io/sistatica/reference/welchOmni.md)
  : Omnibus test for Welch one-way ANOVA
- [`welchPost()`](https://tsuda16k.github.io/sistatica/reference/welchPost.md)
  : Post hoc tests for Welch one-way ANOVA (Games-Howell p-values +
  Welch t-test CIs)
